# CPECLUB C Engineering Projects

This repository contains my solutions for the computer engineering introductory C programming tasks.

## 1. Cramer's Rule Solver (`Linear_Algebra/`)
This program solves a system of linear equations using Cramer's Rule. 
- **How it works:** It takes the coefficients of a 2x2 matrix and calculates the determinant to find the values of X and Y.
- **How to run:** Compile using `gcc solver.c -o solver` and run with `./solver`.

## 2. Mini Calculator (`Calculator/`)
A simple command-line calculator that performs basic arithmetic.
- **How it works:** It prompts the user for two numbers and an operator (+, -, *, /). It includes an edge-case check to prevent division by zero.
- **How to run:** Compile using `gcc calculator.c -o calculator` and run with `./calculator`.

## 3. Even/Odd Logic Gate (`Even_Odd/`)
A concise program that classifies integers.
- **How it works:** It utilizes the modulo operator (`%`) to determine if a number has a remainder when divided by 2.
- **How to run:** Compile using `gcc even_odd.c -o even_odd` and run with `./even_odd`.
